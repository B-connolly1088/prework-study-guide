HTML

Hypertext Markup Language(HTML) is a markup language that your web browser will utilize that allows your browser to show anything you see online.

  1. Tags are always surrounded by angle brackets.
  2. Most tags come in pairs and surround the material they affect.
  3. The first tag is the opening tag.
  4. The second or closing tag always starts with a forward slash.

CSS

  • CSS provides the styling of a webpage.
  • CSS stands for Cascading Style Sheets.
  • CSS describes the presentation of a document written in a markup language.
  • You can assign colors in CSS by including semantic references or by refering to its hexadecimal code.
  • A margin indicates how much space we want around the outside of an element.

Git

  • Git is a content tracker.
  • Git is an open source version control software.

JavaScript

  • JavaScript gives logic to a web application.
  • JavaScript is one of the three cornerstone languages for web development.
